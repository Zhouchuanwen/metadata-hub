Crawler configuration panel

Overview

This is the configuration panel for directly accessing the crawler. Make sure to fill out all forms before hitting the Submit button.

The input directories have to be specified in the following format:

PATH, RECURSIVE ; PATH, RECURSIVE ; ...

For example:

/home/john/Documents, true ; /home/doe/Pictures, false

This would invoke the crawler to process the directory /home/john/Documents recursively and to crawl only the files that are directly located in /home/doe/Pictures
